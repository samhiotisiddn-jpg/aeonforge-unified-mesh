Multi-agent orchestration platform for sovereign AI mesh deployment.

## Features

- **Dependabot Auto-Merge**: Automatically merges Dependabot PRs when tests pass
- **SLSA Provenance**: Supply-chain security attestation for releases
- **Comprehensive Architecture**: See `ARCHITECTURE.md` for full documentation

## Dependencies

This project uses Dependabot for automated dependency updates:

- **pip**: Python dependencies
- **npm**: Node.js dependencies  
- **GitHub Actions**: CI/CD workflow updates

## Automated Dependency Management

Dependencies are automatically kept up-to-date via:
- Weekly Dependabot scans for all package ecosystems
- Auto-merge of PRs when status checks pass
- Grouped updates for related dependencies
